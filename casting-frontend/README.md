# casting-frontend

React frontend for the Paradies casting system.

## Running

```sh
npm install
npm run dev
```

Available at `http://localhost:5173`. API requests to `/castings` are proxied to the backend at `http://localhost:8080`.

## Tech stack

- React 19, TypeScript, Vite
- Tailwind CSS 4, shadcn/ui (Base UI)
- React Router 7
